Different selector types

  • b – tag name selector
  • #id – id selector
  • .myClass – class name selector
  • .myClass.myAnotherClass– several classes selector
  • b i – nested tags selector
  • b > u – child element selector
  • i+b – neighbour element selector
  • u~i – related element selector
  • [href] – attribute selector
  • [href='#'] – attribute value selector

Here we can see how different kinds of selectors interrcat between each other. Maybe it looks a bit messy but with adding some scripts it'll become more clear. And yeah, I've combined 2 parts of the task in 1. Hope it's okay.
